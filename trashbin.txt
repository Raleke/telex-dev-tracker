## Dependencies

- **Runtime**: express, better-sqlite3, axios, cron, dotenv, pino, zod
- **Mastra**: @mastra/core, @mastra/libsql, @mastra/loggers, @mastra/memory
- **Development**: typescript, ts-node, @types/node, @types/express, @types/better-sqlite3, @types/cron
 
## License
 



## Development

- **Build**: `npm run build` - Compiles TypeScript to JavaScript.
- **Development**: `npm run dev` - Runs with ts-node for hot reloading.
- **Testing**: `npm test` - Run tests (currently placeholder).
